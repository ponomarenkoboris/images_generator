## Установка проекта:
Клонировать репозиторий:
```cmd
git clone https://github.com/ponomarenkoboris/images_generator.git && cd images_generator
```

**ИЛИ**

Скачать архив приложения

---
## Работа в приложении

1. Создание образа приложения:
```cmd
docker-compose build
```

2. Запуск контейнера (запуск приложения):
```cmd
docker-compose up
```

3. Открыть браузер и перейти по адрессу:
```
http://localhost:3000
```

5. Остановка работы приложения: 
```
Сочетание клавиш в cmd: CTRL + C 
```
